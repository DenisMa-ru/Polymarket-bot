# 🚀 Polymarket Bot v3.2 - Краткая инструкция по настройке

## ✅ Что добавлено

### 4 новых модуля для разгона депозита:

1. **StateManager** (`src/core/state-manager.ts`)
   - Автосохранение позиций каждые 30 секунд
   - Восстановление после перезапуска
   - История сделок и PnL статистика
   - Встроенный риск-менеджмент

2. **Smart Money V2** (`src/services/smart-money-service-v2.ts`)
   - Динамический размер позиции (пропорционально капиталу кита)
   - Автоматический Take-Profit +30%
   - Автоматический Stop-Loss -15%
   - Trailing Stop 10% для фиксации прибыли

3. **Scalping Service** (`src/services/scalping-service.ts`)
   - Торговля на 5-15 минутных рынках
   - Технические индикаторы (RSI, Volume, MACD)
   - Быстрый выход: TP +15%, SL -10%
   - Цель: 5-8% прибыли в день

4. **Telegram Notifications** (`src/services/telegram-service.ts`)
   - Уведомления о всех сделках
   - Оповещения о TP/SL
   - Ежедневные отчёты
   - Предупреждения о рисках

## 🛠️ Быстрый старт (5 минут)

### 1. Установите зависимости
```bash
cd C:\Users\Денис\Downloads\Polymarket-bot-main
npm install
```

### 2. Настройте Telegram (опционально)

**Получить токен бота:**
1. Откройте @BotFather в Telegram
2. Отправьте `/newbot`
3. Следуйте инструкциям
4. Скопируйте токен

**Получить Chat ID:**
1. Откройте вашего бота
2. Отправьте любое сообщение
3. Откройте @userinfobot
4. Скопируйте Chat ID

**Обновите .env:**
```env
TELEGRAM_ENABLED=true
TELEGRAM_BOT_TOKEN=ваш_токен_сюда
TELEGRAM_CHAT_ID=ваш_chat_id_сюда
```

### 3. Запустите бота
```bash
npx tsx bot-with-dashboard.ts
```
